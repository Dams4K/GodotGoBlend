# Godot GoBlend

A Godot plugin to help me import gltf files created by [Blender GoBlend](https://github.com/Dams4K/BlenderGoBlend)

## How to use

Use the file `res://addons/godot_goblend/goblend_scene_post_import.gd` as the post import file.

> [!CAUTION]
> This is not intended to be used by people other than me. Not because I'm selfish, it's just that I've made this tool to suit my needs.
